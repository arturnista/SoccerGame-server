Games

NW Game

Online Battle Arena with a Smash Bros like mechanic, where your objective is to throw your enemies out of the arena using your spells.

The Backend of the game is entirely build in Node.js, using socket.io for the client-server communication. The physics engine was entirely build by me, as a self challenge.

The site was built using React and PIXI.js renders the game. The art was also done by me.

Destruction chain

Android Game developed on Unity 3D, a top down spaceship shooting game, with the differential that one enemy hit causes a chain reaction, creating more projectiles every consecutive hit.

The game was built using C# as the main language.
