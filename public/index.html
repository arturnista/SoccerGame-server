Games

NW Game

Online Battle Arena with a Smash Bros like mechanic, where your objective is to throw your enemies out of the arena using your spells.

The Backend of the game is entirely build in Node.js, using socket.io for the client-server communication. The physics engine was entirely build by me, as a self challenge.

The site was built using React and PIXI.js renders the game. The art was also done by me.

Lady of the Colors

Platform with puzzles. You can fly to avoid obstacles and get close to your objective.

Game developed in Unity 3D. Game developed with more focus on game and level design.

Also, the code is developed with a Composition thinking, becoming very simple to add new features.

Destruction chain

Android Game developed on Unity 3D, a top down spaceship shooting game, with the differential that one enemy hit causes a chain reaction, creating more projectiles every consecutive hit.

The game was built using C# as the main language.

C Jumper

Casual Game, developed on Unity 3D. You jump obstacles and collect diamonds.

You can play in different enviroments with differents challenges and obstacles.

The game was built using C# as the main language.

FPS

Web Thresh Game

Simple game about collecting souls. Skill is needed to throw your hook and hit a soul.

Developed in a single weekend, with a small set of modifications done after that.

This game was done with a team of three people, with the aim to improve our code skills.

Soccer game

One of my first games developed from start to end. Still, the end result is pretty satisfactory.

Game of life
